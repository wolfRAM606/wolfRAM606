# Hey, I'm Sriram

I'm a CS undergrad who likes building things that solve actual problems — not just toy demos, but systems that work in real environments. Most of my projects come from genuine curiosity about how things connect under the hood: how a web UI talks to a remote server, how a neural network learns to identify a raga, or how a campus community can be brought together with a simple app.

---

### What I've been working on

**[DRDL Remote HPC Job Execution System](https://github.com/wolfRAM606/drdl-remote-hpc-job-execution-prototype)** — A web-based system for submitting and monitoring computational jobs on remote Linux servers over SSH. Built during my internship at DRDL, DRDO. Features PBS/TORQUE-compatible scheduling, real-time monitoring, and intelligent error diagnostics. Python, Streamlit, Paramiko.

**[Raga Lens](https://github.com/wolfRAM606/raga-lens)** — AI-powered Carnatic raga detection using a DNN/CNN ensemble trained on 96 ragas. FastAPI backend, React frontend, TensorFlow models. Achieved ~70% accuracy on DNN with 96-dimensional MFCC+chroma+spectral features.

**[Campus Homie](https://github.com/wolfRAM606/Campus-Homie)** — A lightweight campus support platform built with React and Firebase. Students and faculty can post help requests, browse by category, and connect with each other. Simple idea, clean execution.

**[Recipeek](https://github.com/wolfRAM606/Recipeek---Usability)** — A mobile-first recipe app UI prototype built for usability testing. Six interactive screens covering recipe discovery, meal planning, and inventory management. Pure HTML/CSS/JS.
